Code accompanying the thesis:

**Rooftop Photovoltaic Panel Segmentation in Aerial Images: Efficient, Transformer-Based, and Few-Shot Deep Learning Approaches**

## Overview

This repository contains the code associated with the thesis work on rooftop photovoltaic (PV) panel segmentation in aerial images. The thesis investigates three connected deep learning directions:

1. Efficient attention-enhanced UNet for rooftop PV segmentation  
2. Transformer-based UNet for rooftop PV segmentation  
3. Few-shot DINOv2-based rooftop PV segmentation  

## Repository Contents

- `efficient-unet.ipynb` — code for the efficient attention-enhanced UNet work
- `pv_seg.ipynb` — code for the Transformer-based UNet work
- `A1.ipynb` to `A5.ipynb` — ablation study notebooks for the Transformer-based UNet work
